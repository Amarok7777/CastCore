Willkommen bei CastCore!
3 kurze Schritte, dann bist du startklar.
1
Twitch verbinden
Klicke unten auf „Mit Twitch einloggen" → ein Code erscheint → gib ihn auf twitch.tv/activate ein.
Danach werden Subs, Bits und Raids automatisch erkannt.
2
YouTube-Kanal eintragen (optional)
Trage deinen Kanalnamen oder Handle ein (z.B. @meinkanal) und klicke auf „Aktiven Stream suchen" wenn du live bist. Superchats und Memberships werden dann automatisch angezeigt.
3
OBS Browser Source einrichten
Öffne ein Tool (z.B. EventForge für Alerts) → kopiere die Overlay-URL → füge sie in OBS als Browser Source ein. Fertig.

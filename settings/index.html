SplitFlow — Hotkeys
Timer-Aktionen
Start / Split / Fortsetzen
Startet, setzt Split oder setzt nach Pause fort
Pause / Fortsetzen
Pausiert den laufenden Timer
Reset
Timer und alle Splits zurücksetzen
Undo Split
Letzten Split rückgängig machen
Split überspringen
Aktuellen Split ohne Zeit überspringen
Klicke einen Button, dann drücke die gewünschte Taste. Escape bricht ab. Hotkeys wirken systemweit — auch wenn das Spiel im Vordergrund ist.
Unterstützte Tasten
Numpad 0–9 F1–F12 Space Insert Delete Home / End

Numpad-Tasten sind ideal — sie sind beim Spielen selten belegt.

Konflikt-Erkennung

Wenn zwei Aktionen die gleiche Taste haben, werden die Buttons rot markiert und das Speichern wird blockiert.

Twitch — App-Konfiguration

Die Client ID wird für den Twitch-Login benötigt. Ohne sie ist kein OAuth-Login möglich.
Eigene App erstellen unter dev.twitch.tv/console → Neue App → Redirect URL: http://localhost

Wo bekomme ich die Client ID?
  1. dev.twitch.tv/console aufrufen
  2. „Anwendung registrieren"
  3. Name beliebig, Kategorie: „Chat Bot"
  4. OAuth-Redirect: http://localhost
  5. Client ID kopieren und hier einfügen
